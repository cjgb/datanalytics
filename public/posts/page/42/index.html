Un generador de datos sintéticos para proteger la privacidad de los microdatados

2018-5-23
consultoría, estadística
DataSynthesizer (véase también el correspondiente artículo) es un programa en Python que: 1. Toma una tabla de datos (microdatos, de hecho) que contiene información confidencial. 2. Genera otra _aleatoria_ pero que conserva (¿los conservará?) la estructura básica de la información subyacente (conteos, correlaciones, etc.). Está pensado para poder realizar el análisis estadístico de (determinados) datos sin verlos propiamente. Particularmente interesante es el algoritmo para preservar la correlación entre columnas.
